# Headings in Markdown

> Learn how to organize your Markdown documents using headings.

---

# What are Headings?

Headings are used to create titles and organize your content into sections. They make your document easier to read and navigate.

Markdown provides **six levels of headings**, from the largest (`#`) to the smallest (`######`).

---

# Heading Levels

| Level | Syntax |
|------|--------|
| Heading 1 | `# Heading 1` |
| Heading 2 | `## Heading 2` |
| Heading 3 | `### Heading 3` |
| Heading 4 | `#### Heading 4` |
| Heading 5 | `##### Heading 5` |
| Heading 6 | `###### Heading 6` |

---

# Syntax

To create a heading, place one or more `#` symbols before the heading text.

```md
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```

> **Note:** Always leave a space between the `#` symbol and the heading text.

Correct:

```md
# Python
```

Incorrect:

```md
#Python
```
