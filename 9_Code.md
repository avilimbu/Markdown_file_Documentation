# Code in Markdown

Markdown allows you to display code using **inline code** or **code blocks**.

## Inline Code

Use a single backtick (<code>`</code>) to highlight a word or a short piece of code.

### Markdown

`` `print("Hello")` ``

### Output

`print("Hello")`

---

## Code Block

Use three backticks (<code>```</code>) before and after the code block.

### Markdown

````md
```python
print("Hello, World!")
```
