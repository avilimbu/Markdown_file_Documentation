# Code in Markdown

Markdown allows you to display code in a clean and readable format. It supports two types of code formatting:

- Inline Code
- Code Blocks

---

## Inline Code

Inline code is used to highlight a word, command, variable, or a short piece of code within a sentence.

### Syntax

Use a **single backtick (`)** before and after the text.

### Markdown

`` `print("Hello, World!")` ``

### Output

`print("Hello, World!")`

### Example

```md
Use the `print()` function to display output.
```

## Code Blocks

Code blocks are used to display multiple lines of code.

### Syntax

Use **three backticks (```)** before and after the code.

### Markdown

````md
```
print("Hello, World!")
print("Welcome to Markdown")
```
