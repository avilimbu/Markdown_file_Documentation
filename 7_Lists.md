# Lists in Markdown

Lists are used to organize information in a clear and readable format. Markdown supports **unordered**, **ordered**, and **nested** lists.

## Unordered List

Use `-`, `*`, or `+` followed by a space.

### Markdown

```md
- Apple
- Banana
- Mango
```

### Output

- Apple
- Banana
- Mango

---

## Ordered List

Use numbers followed by a period (`.`).

### Markdown

```md
1. Python
2. Java
3. C++
```

### Output

1. Python
2. Java
3. C++

---

## Nested List

Indent a list item using **2 or 4 spaces** to create a sublist.

### Markdown

```md
- Fruits
  - Apple
  - Mango
- Vegetables
  - Carrot
```

### Output

- Fruits
  - Apple
  - Mango
- Vegetables
  - Carrot

---

## Best Practices

- Use unordered lists when the order doesn't matter.
- Use ordered lists for steps or instructions.
- Keep list items short and consistent.

---

## Summary

Markdown supports three types of lists:

| List Type | Syntax |
|-----------|--------|
| Unordered | `-`, `*`, `+` |
| Ordered | `1.`, `2.`, `3.` |
| Nested | Indent with 2 or 4 spaces |
