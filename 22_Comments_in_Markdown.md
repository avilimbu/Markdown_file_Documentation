# Comments in Markdown

Comments are used to add notes or reminders that are **not displayed** in the final rendered document.

> **Note:** Markdown does not have its own comment syntax. Instead, HTML comments are used.

---

## Syntax

Use the HTML comment tag.

### Markdown

```html
<!-- This is a comment -->
```

### Output

The comment is **not visible** in the rendered document.

---

## Example

### Markdown

```html
# My Project

<!-- Update this section before publishing -->

This project demonstrates the basics of Markdown.
```

### Output

# My Project

This project demonstrates the basics of Markdown.

---

## Best Practices

- Use comments for notes, reminders, or instructions.
- Do not include sensitive or confidential information in comments.
- Remove unnecessary comments before publishing if they are no longer needed.

---

## Summary

Comments help you leave hidden notes in your Markdown files using HTML comment tags.
