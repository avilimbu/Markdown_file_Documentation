# Highlight Text in Markdown

Highlighted text is used to draw attention to important words or sentences. Some Markdown editors support this feature using double equal signs (`==`).

> **Note:** Highlight text is **not part of standard Markdown** and is **not supported by GitHub Flavored Markdown (GFM)**. It is supported by editors like **Obsidian**, **Typora**, and some Markdown processors.

---

## Syntax

Use double equal signs (`==`) before and after the text.

### Markdown

```md
==This text is highlighted.==
```

### Output

==This text is highlighted.==

---

## Example

### Markdown

```md
Please read the ==Important Note== before continuing.

Python is ==easy to learn== for beginners.
```

### Output

Please read the ==Important Note== before continuing.

Python is ==easy to learn== for beginners.

---

## Best Practices

- Highlight only important words or sentences.
- Avoid overusing highlighted text.
- Make sure your Markdown editor supports this feature.

---
