# Blockquotes in Markdown

A **blockquote** is used to highlight quotes, notes, or important information. It is created using the greater-than symbol (`>`).

## Syntax

```md
> This is a blockquote.
```

### Output

> This is a blockquote.

---

## Multiple Lines

You can create a blockquote with multiple lines by adding `>` before each line.

### Markdown

```md
> Markdown is simple.
> It is widely used for documentation.
```

### Output

> Markdown is simple.
> It is widely used for documentation.

---

## Nested Blockquote

Use multiple `>` symbols to create nested blockquotes.

### Markdown

```md
> Main Quote
>> Nested Quote
```

### Output

> Main Quote
>> Nested Quote

---

## Best Practices

- Use blockquotes for quotes, notes, or important information.
- Keep blockquotes short and easy to read.
- Use nested blockquotes only when necessary.

---

## Summary

Blockquotes help emphasize important text or quotations. Use the `>` symbol for a blockquote and `>>` for a nested blockquote.
