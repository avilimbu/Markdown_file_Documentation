# Line Breaks in Markdown

> Learn how to move text to a new line without starting a new paragraph.

---

# What is a Line Break?

A **line break** moves the text to the next line **without creating a new paragraph**.

This is useful when writing addresses, poems, lyrics, or any content where each line should appear separately.

---

# Methods to Create a Line Break

Markdown provides two common ways to create a line break:

1. Add **two spaces** at the end of a line.
2. Use the HTML `<br>` tag.

---

# Method 1: Two Trailing Spaces

Add **two spaces** at the end of a line before pressing **Enter**.

### Markdown

```md
Hello World.  
Welcome to Markdown.
```

### Output

Hello World.  
Welcome to Markdown.

> **Note:** The two spaces are invisible, so they can be easy to miss.

---

# Method 2: HTML `<br>` Tag

You can also use the HTML `<br>` tag.

### Markdown

```md
Hello World.<br>
Welcome to Markdown.
```

### Output

Hello World.<br>
Welcome to Markdown.

The `<br>` tag is more visible and is commonly used in many Markdown editors.

---

# Example

### Markdown

```md
Name: John Doe<br>
Email: john@example.com<br>
Phone: +977-9800000000
```

### Output

Name: John Doe<br>
Email: john@example.com<br>
Phone: +977-9800000000

---

# Best Practices

- Use a **blank line** to create a new paragraph.
- Use a **line break** only when the text should continue in the same paragraph.
- Prefer the `<br>` tag when you want your Markdown to be more readable.

---

# Summary

A line break moves text to the next line without starting a new paragraph. You can create one by adding **two spaces** at the end of a line or by using the HTML `<br>` tag.
