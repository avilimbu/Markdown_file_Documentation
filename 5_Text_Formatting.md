# Text Formatting in Markdown

Markdown provides simple symbols to format text, making important words or phrases stand out.

## Bold

Use **double asterisks (`**`)** or **double underscores (`__`)** to make text bold.

### Markdown

```md
**Bold Text**
__Bold Text__
```

### Output

**Bold Text**  
__Bold Text__

---

## Italic

Use a **single asterisk (`*`)** or **single underscore (`_`)** to italicize text.

### Markdown

```md
*Italic Text*
_Italic Text_
```

### Output

*Italic Text*  
_Italic Text_

---

## Bold and Italic

Use **three asterisks (`***`)** to make text both bold and italic.

### Markdown

```md
***Bold and Italic***
```

### Output

***Bold and Italic***

---

## Strikethrough

Use **double tildes (`~~`)** to strike through text.

### Markdown

```md
~~Deleted Text~~
```

### Output

~~Deleted Text~~

---

## Best Practices

- Use **bold** to highlight important words.
- Use *italic* for emphasis or titles.
- Use ~~strikethrough~~ to show removed or outdated text.
- Avoid excessive formatting to keep your document clean.

---

## Summary

Markdown makes text formatting simple with a few symbols:

| Formatting | Syntax |
|------------|--------|
| **Bold** | `**Text**` or `__Text__` |
| *Italic* | `*Text*` or `_Text_` |
| ***Bold & Italic*** | `***Text***` |
| ~~Strikethrough~~ | `~~Text~~` |
